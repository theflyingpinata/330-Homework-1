Description

While watching anime one day, we decided to model out Rich Media project after an anime fight. Specifically, the way fighting entities circle and dance around each other during battle. We implemented seek and collision functions so the balls could interact with each other, as well as a state machine that changes the behaviors of the balls whenever they collide with each other.

Instructions

  • Using the size sliders, you can change how large the balls are
  • Click the dropdown to change the global alpha value, which affects the trails the circular objects leave when they move
  • Change the action weights to change how often the orbs enter a specific state
    • Attack: Ball will seek the other ball
    • Dash: Ball will move fast in it's current direction
    • Guard: Ball will slow down
  • Use the Ball Color Picker to change the color of the balls during a specific state
